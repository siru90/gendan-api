- ` 192.168.1.15:7880/api/tracking_order/express/channel/list `

##### 请求方式

- GET

##### 参数

| 参数名  | 必选 | 类型  | 说明          |
|:-----|:---|:----|-------------|
| page | 否  | int | 页码, 默认值:1   |
| size | 否  | int | 页大小, 默认值:20 |
