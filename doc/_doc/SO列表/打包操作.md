##### 简要描述

- 总务客服操作

##### 请求URL

- ` 192.168.1.15:7880/api/tracking_order/so/db_op `

##### 请求方式

- POST

##### 参数

| 参数名       | 必选 | 类型     | 说明               |
|:----------|:---|:-------|------------------|
| so_id     | 是  | int    | SO ID            |
| db_status | 是  | int    | 状态： 0 打包中 1 打包完毕 |
| note      | 是  | string | 备注               |

##### 返回示例

```
{
    "code": 0,
    "data": {
        "affected": 1
    },
    "time": 0.131
}
```

##### 返回参数说明

| 参数名      | 类型  | 说明                |
|:---------|:----|-------------------|
| affected | int | affected > 0 代表成功 |
