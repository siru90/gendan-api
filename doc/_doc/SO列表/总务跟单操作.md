##### 请求URL

- ` 192.168.1.15:7880/api/tracking_order/so/gd_op `

##### 请求方式

- GET

##### 参数

| 参数名                       | 必选 | 类型     | 说明               |
|:--------------------------|:---|:-------|------------------|
| so_id                     | 是  | int    | 发货单号             |
| check_status              | 是  | int    | 缺货/货齐 状态 0缺货 1货齐 |
| note                      | 是  | string | 备注               |
| shipping_way              | 否  | string | 发货方式             |
| ship_task_delivery_number | 否  | string | 发货任务单号           |

##### 返回示例

```
{
    "code": 0,
    "data": {
        "affected": 3
    },
    "time": 1.673
}
```

##### 返回参数说明
