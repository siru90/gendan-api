##### 简要描述

- SO操作日志列表

##### 请求URL

- ` 192.168.1.15:7880/api/tracking_order/so/logs `

##### 请求方式

- GET

##### 参数

| 参数名   | 必选 | 类型  | 说明          |
|:------|:---|:----|-------------|
| so_id | 是  | int | SO ID       |
| page  | 是  | int | 页码，默认：1     |
| size  | 是  | int | 页大小，默认：1000 |
